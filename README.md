## Run Locally

Clone the project

```bash
  git clone https://github.com/AT74PH0L/Healthy-Vibes-demo.git
```

Go to the project directory

```bash
  cd Healthy-Vibes-demo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
